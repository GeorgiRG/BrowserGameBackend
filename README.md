# BrowserGameBackend

.NET 7 application that exposes REST APIs for https://github.com/GeorgiRG/BrowserGame \
Built with ASP.NET and Entity Framework

**Features done:**\
-Authentication\
-Registration + email confirmation\
-Login and sessions
